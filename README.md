# Driving-License-Management-System

C#/.NET Driving License Management System using SQL Server, featuring CRUD operations and a 3-tier architecture.
A backend-focused Windows application built with C#, .NET Framework, and SQL Server.

## 🚀 Features
- CRUD operations
- Database design with relationships
- Input validation
- Error handling

## 🧠 Tech Stack
- C#
- .NET Framework (Windows Forms)
- SQL Server
- ADO.NET

## 📊 Architecture
This project follows a 3-tier architecture:
- Presentation Layer 
- Business Logic Layer 
- Data Access Layer

## 🗄️ Database
The database is designed using an ERD with proper relationships and constraints.
